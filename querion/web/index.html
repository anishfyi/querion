What Querion does

Plug in any platform

Add a key and secret to the environment, drop in the API docs, and connect your Postgres. No code. Querion reads your schema and your endpoints and starts answering.

Multi-source reasoning

It joins history in Postgres with live truth from your APIs in a single answer - the way an analyst would, picking the right source per question.

Read-only by contract

SELECT/WITH only, GET only with a per-source allowlist, and a firewall that refuses any request to change data. It never writes anything, anywhere.

Charts and transparency

Trend, comparison, and share-of-whole charts, plus the exact query behind every number so you can trust and reuse it.

Runs on Claude Code CLI

No API key. Querion shells out to your locally authenticated Claude Code CLI, with Opus recommended for the deepest reasoning.

Semantic layer via Trove

Pair it with Trove to auto-build a semantic layer - your metric definitions and domain vocabulary - so answers speak your language.
